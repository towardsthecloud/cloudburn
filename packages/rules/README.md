# @cloudburn/rules

Pure rule declarations and types for Cloudburn cost optimization checks.

This package contains the rule definitions (`Rule`, `Finding`, `Severity`, `ScanMode`) used by the Cloudburn scan engine. It has no I/O, no AWS SDK calls, and no engine logic.

## Installation

```sh
npm install @cloudburn/rules
```

## Usage

```ts
import { rules } from "@cloudburn/rules";
```
